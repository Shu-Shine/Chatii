# Chatii
A real-time chat application built with the MERN stack and Socket.io. 

## Tech Stack
- __MERN stack (MongoDB, Express.js, React.js, and Node.js)__  
- __Socket.io__  
- __Bcrypt__  
- __Styled Components__  
- __Docker Compose__  

## Basic Features
* User Authentication
* Encrypted storage
* Avatar Generation
* Real-time Chatting
* Emoji Picker
* Public Chat Room
