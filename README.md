# Chatii

A real-time chat application built with the MERN stack and Socket.io.

## Tech Stack

- **MERN stack (MongoDB, Express.js, React.js, and Node.js)**
- **Socket.io**
- **Bcrypt**
- **Styled Components**
- **Docker Compose**

## Basic Features

- User Authentication
- Encrypted storage
- Avatar Generation
- Real-time Chatting
- Emoji Picker
- Public Chat Room
